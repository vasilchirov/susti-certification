# susti-certification

## Requirements

- Node.js 18+ (or newer)

## Run the project

1. Install backend dependencies:
	- `cd backend`
	- `npm install`

2. Start backend (API on `http://localhost:3000`):
	- `npm run dev`

3. In a new terminal, install frontend dependencies:
	- `cd frontend`
	- `npm install`

4. Start frontend (website on `http://localhost:5000`):
	- `npm start`

5. Open `http://localhost:5000` in your browser.

## Notes

- Backend does not have a `start` script. Use `npm run dev`.
